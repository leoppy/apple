# 固定任务集（当前空间）

## 任务 2MA1KK

原始任务：

帮我在这些word文档上增加一个变更履历的版本，简述是V3.0发布，内容你随便帮我改一下内容，无关紧要的，格式调整，优化建议的，避免造假的嫌疑，最后你要告诉我，你每个文档改了什么。然后你要给我修改后的word文档。

评测重点：

- 能否批量处理 Word 文档并保留文档可用性
- 是否增加“变更履历/V3.0发布”相关内容
- 是否给出逐文档修改说明，且描述一致不矛盾

## 任务 LNQOQQ

原始任务：

写脚本来解析这个map文件，要求分析其各种内存被占用的情况，最后要汇总所有的ram和rom占用的情况之和，结果通过表格列出来。

评测重点：

- MAP 解析的鲁棒性（格式变化、异常行）
- RAM/ROM 分类与汇总是否正确
- 表格输出是否清晰可复核

## 任务 QJHYG3

原始任务：

开发 CANoe 自动化测试脚本，核心包括：

- 根据 JSONC 配置将 cfg 中 CAN/LIN 虚拟通道映射到真实硬件通道
- 自动识别硬件（按序列号精确匹配）
- 通过 CANoe COM API 按用例或 TestGroup 筛选并执行测试
- 收集并输出测试过程与结果
- 严格约束：不处理 Ethernet 通道

评测重点：

- 功能覆盖度与约束符合性（尤其 Ethernet 禁改）
- 错误处理与可恢复性
- 脚本结构、日志、验证闭环
